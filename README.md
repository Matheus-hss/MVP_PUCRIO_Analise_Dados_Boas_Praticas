📊 Análise Exploratória de Dados – Desempenho de Estudantes

Este projeto apresenta uma análise exploratória de dados (EDA) aplicada a um dataset de desempenho de estudantes, com foco em compreender padrões, relações entre variáveis e possíveis fatores que influenciam o desempenho acadêmico.

🎯 Objetivo

O principal objetivo deste projeto é analisar os dados de estudantes e identificar insights relevantes por meio de:

Estatísticas descritivas
Visualizações gráficas
Identificação de padrões e tendências
Preparação inicial dos dados para possíveis modelos de Machine Learning

🧠 Tipo de Problema

📌 Aprendizado supervisionado (regressão)

Variável alvo: desempenho final dos estudantes

📁 Dataset

O dataset utilizado contém informações sobre:

Hábitos de estudo
Frequência
Fatores socioeconômicos
Indicadores acadêmicos

Essas variáveis permitem analisar possíveis correlações com o desempenho dos alunos.

🔍 Etapas do Projeto

1. Definição do Problema
Contextualização do problema
Levantamento de hipóteses
Entendimento dos dados
2. Análise Exploratória
Estatísticas descritivas
Verificação de valores faltantes
Análise de distribuições
Visualização de variáveis individuais e combinadas
3. Visualização de Dados
Uso de gráficos para identificação de padrões
Interpretação dos resultados obtidos
4. Pré-processamento
Tratamento de dados (quando necessário)
Transformações e ajustes
Preparação para etapas futuras de modelagem

📈 Principais Insights
Identificação de padrões de estudo relacionados ao desempenho
Observação de distribuições e possíveis outliers
Relações entre variáveis comportamentais e notas finais
